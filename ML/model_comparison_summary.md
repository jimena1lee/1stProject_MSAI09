# 어린이 보호구역 사고 발생건수 예측 모델 비교 결과 요약

> **데이터**: `seongnam_newlabel.csv` (성남시 어린이 보호구역 110개소)
> **목표**: 사고 발생건수(0~38건) 예측에 적합한 모델 탐색
> **비교 모델**: 선형회귀, 이진분류(RandomForest), Poisson, ZIP, ZINB

## 1. 데이터 기초 탐색 (STEP 0)

**발생건수 분포 특징**
- 57% 이상이 0건으로 영 과잉(zero-inflated) 구조가 명확함
- 비영 샘플은 대부분 3~7건에 집중되나 최대 38건의 긴 꼬리(heavy tail) 존재
- 과산포 지수 12.9 → **강한 과산포** → ZINB가 이론적으로 가장 적합

## 4. 모델별 학습 결과

### 모델 1 | 선형회귀 (대조군)
- R² = **-0.158** → 음수, 예측 성능 거의 없음
- 음수 예측값 발생 (0으로 클리핑 처리)
- 발생 있는 샘플은 100% 감지하나 0건을 0으로 예측하는 능력이 매우 낮음 (10%)
- 카운트 데이터 특성상 구조적으로 부적합

### 모델 2 | 이진분류 - RandomForest (대조군)
- Accuracy **72.2%**, Macro F1 **0.714**, AUC-ROC **0.813**
- 0건 정확도 **80%**, 비영 정확도 **62.5%**
- 발생 여부는 어느 정도 판별하나 실제 건수 예측 불가 (본질적 한계)

### 모델 3 | Poisson 회귀
- 수렴 성공
- AIC = 362.67 (ZIP/ZINB 대비 크게 높음)
- 0건 정확도 낮음 (20%) → 영 과잉 구조를 전혀 반영 못함
- **유의한 피처**: 무인교통단속카메라(+), 생활안전CCTV(+), 신호등(+), 횡단보도(-), 어린이 비율(%)(-)

### 모델 4 | Zero-Inflated Poisson (ZIP)
- 수렴 성공 (bfgs)
- Log-Likelihood = -101.95, Pseudo R² = 0.437
- **유의한 피처 (카운트 파트)**: 도로안전표지(-), 무단횡단방지펜스(-), 생활안전CCTV(+), 신호등(+), 옐로카펫(+), 어린이 비율(%)(-)
- 인플레이션 파트: 어린이 비율(%)이 유의 (p=0.026)

### 모델 5 | Zero-Inflated Negative Binomial (ZINB)
- 수렴 성공 (bfgs)
- Log-Likelihood = -100.74, Pseudo R² = 0.261
- 과산포 파라미터 α = 0.0714
- **유의한 피처 (카운트 파트)**: 도로안전표지(-), 무단횡단방지펜스(-), 신호등(+), 옐로카펫(+), 어린이 비율(%)(-)
- 인플레이션 파트: 어린이 비율(%)이 유의 (p=0.028)

## 8. 종합 결론

### 모델 선택
> **ZINB(Zero-Inflated Negative Binomial)** 가 이 데이터에 가장 적합한 모델

- 과산포 지수 12.6의 강한 과산포 + 57% 영 과잉 구조를 동시에 반영
- AIC 기준 ZIP 대비 소폭 우수, Poisson 대비 압도적 우수
- Test MAE 2.82 → val 대비 개선 (과적합 없음)

### 주요 인사이트
1. **안전 시설 효과**: 도로안전표지, 무단횡단방지펜스는 유의미한 사고 감소 효과
2. **교통량 혼재 효과**: 신호등, 횡단보도는 양의 계수 → 이는 시설이 나쁜 것이 아니라 원래 교통량이 많은 지점에 설치되기 때문
3. **어린이 비율(%)**: 음의 계수 → 어린이 비율이 높은 주거 지역보다 통학로(어린이 비율 낮고 통행량 많은 지점)에서 사고 多

### 한계 및 개선 방향
- 샘플 수 110개로 학습 데이터 70건은 ZIP/ZINB의 안정적 추정에 충분하지 않을 수 있음
- 비영 정확도 60%: 사고 발생 시 건수 예측 정밀도 개선 여지 있음
- 공간적 자기상관(인근 보호구역 간 영향) 미반영 → 공간 회귀 모델 도입 고려
